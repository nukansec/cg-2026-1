# Computação Gráfica - UFG

Este repositório é dedicado a armazenar todos os códigos, exercícios e projetos desenvolvidos ao longo da disciplina de **Computação Gráfica**, do curso de Ciência da Computação na Universidade Federal de Goiás (UFG).

## 📌 Sobre o Repositório
O objetivo principal é registrar a evolução contínua do aprendizado durante o semestre. O repositório funcionará como um portfólio da disciplina, começando pelos fundamentos de desenho em tela (Canvas com JavaScript) até a implementação de algoritmos mais avançados de renderização e transformações geométricas.

## 📂 Conteúdo
Ao longo do semestre, este espaço será atualizado com:
- Atividades práticas e desenhos livres.
- Implementação de algoritmos clássicos (traçado de retas, circunferências, preenchimento, etc.).
- Evolução da biblioteca gráfica construída passo a passo nas aulas.
---
## 🤖 Usos de IA
Para garantir a transparência acadêmica e seguir as diretrizes da disciplina:
- **Tutoria e Conceitos:** Ferramentas de IA foram utilizadas como suporte para esclarecer dúvidas teóricas sobre a API do Canvas, lógica de eventos de teclado e álgebra linear aplicada.
- **Documentação:** Este arquivo `README.md` foi estruturado e editado com o auxílio de IA.
- **Integridade do Código:** A IA não foi utilizada para a geração automática de códigos de implementação, servindo apenas como guia para o entendimento das funções e depuração de lógica.
